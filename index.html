Sobre Velarion

Una plataforma, no un bot aislado

Velarion Platform es una plataforma tecnológica mexicana enfocada en el desarrollo de sistemas de automatización, asistentes inteligentes, comunicación empresarial y soluciones digitales. Su objetivo es crear herramientas que permitan a las organizaciones atender mejor a sus clientes, reducir tareas repetitivas y mantener procesos más ordenados, medibles y escalables.

Uso responsable de mensajería

Comunicación iniciada por el cliente

Twilio forma parte de la infraestructura tecnológica considerada para el envío de comunicaciones relacionadas con reservaciones iniciadas por el propio cliente.

  • El cliente proporciona voluntariamente su número telefónico durante la llamada.
  • Antes de recibir un mensaje, se le informa que llegará una confirmación por WhatsApp.
  • Los mensajes pueden incluir confirmaciones, cancelaciones, actualizaciones y recordatorios.
  • No se utiliza el sistema para spam ni mensajes masivos no solicitados.
  • No se venden números telefónicos ni datos personales.
  • El usuario puede solicitar en cualquier momento la cancelación de comunicaciones.
  • Las comunicaciones están relacionadas únicamente con el servicio solicitado.
Seguridad y privacidad

Principios que seguimos

  • Recopilación limitada de datos: solo lo necesario para prestar el servicio solicitado.
  • Uso de la información únicamente para los fines informados.
  • Protección de credenciales y control de acceso a los sistemas.
  • Separación entre datos de prueba y datos reales de clientes.
  • No se venden datos personales a terceros.
  • Posibilidad de solicitar la corrección o eliminación de datos cuando corresponda.
  • Uso de servicios tecnológicos de terceros para procesar comunicaciones de forma segura.

Ningún sistema es absolutamente invulnerable; trabajamos con buenas prácticas de seguridad de forma continua.
